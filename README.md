# Event Search Multi-Platform Project

A full-stack event discovery ecosystem that allows users to search, filter, and favorite events worldwide. This project was developed as a comprehensive exercise in cross-platform development, utilizing the **Ticketmaster API** for data and **Google Maps API** for location services.

* **Event Discovery:** Real-time search for music, sports, and arts using the Ticketmaster API.
